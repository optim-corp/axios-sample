## Overview
* This project is a sample to Application that performs Restful API communication between front end and back end server.
* Stacks
	* Schema-First Development
	* OpenAPI 3
	* Vue.js+TypeScript+Axios
	* Java+SpringBoot

## Requirement
* Jdk 1.8
* Maven 3.6.0
* Node >= v6.5.0
* yarn >= 1.13.0
* Vue CLI >= v3.5.0
* Git, Docker
